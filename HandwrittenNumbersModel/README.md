Dieses Projekt nutzt ein mit Scikit-Learn trainiertes Machine-Learning-Modell zur Erkennung handgeschriebener Ziffern. Über eine integrierte Tkinter-GUI können Benutzer Zahlen von 0 bis 9 zeichnen, die das Modell anschließend in Echtzeit analysiert und klassifiziert. Ein kompaktes Beispiel für die Verarbeitung von Bilddaten und die Implementierung von ML-Modellen in Desktop-Anwendungen. 

Die Genauigkeit könnte verbessert werden, wenn ich eigene Trainingsdaten verwenden würde, da mein Input sich leicht unterscheidet von den Daten aus dem Trainingsset.

Bei den Eingaben hilft es, wenn man versucht möglichst die gesamte Fläche zu benutzen.
